## Getting Started

```bash
# install dependencies
pnpm install
# run the development server
pnpm dev
```

## Features

#### [`Pnpm`](https://pnpm.io/)

- It installs all dependencies into a single central store.
- Provides a faster experience.
- Results in a more readable and less conflicting lock file.
